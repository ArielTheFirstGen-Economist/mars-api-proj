# mars-api-proj

In this project, I create a graph using R and R Studio, then build a website to display the Bitcoin data over time.

You can download these files and run them yourself. To run the files please use the following options:

OPTION 1:

Use VS Code with the R extension and R interactive terminal to run the R file.

OPTION 2:

First, download R AND R Studio to run the R file. Once you have saved the interactive graph as an HTML widget, you can run the HTML, CSS, JS as normal.
